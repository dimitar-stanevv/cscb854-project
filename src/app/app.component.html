What's in the box?

This project is made as a demonstration of what a simple landing page and/or portfolio could look like in 2022. For a quick overview of the tech stack, see the cards below:

Serverless deployment

Netlify allows us to seamlessly and automatically deploy the project directly from a GitHub repository upon a commit, for example. Going "serverless" can be categorized as a fairly modern approach. Simple lambda functions can also be used with Netlify for backend tasks such as authentication.

How does it work?
Modern framework

Angular is used in this project. It's one of the more capable frontend frameworks available today. It uses TypeScript which is a superset of JavaScript and is now almost 10 years old. As such it's very mature and is quite opinionated, unlike React.js, for example.

Built to impress

As the web is becoming more and more sophisticated and congested with web pages, it's more important than ever for designs to stand out. This project incorporates Sass as its style preprocessor and is built around a reusable consistent custom-made theme.

Ready for the modern web.
And for modern devices, too.

This project has been optimized to work on any Internet-enabled device. The web is the only truly universal platform and as such modern websites need to support a variety of devices, such as phones, tablets, laptops & desktops - above 60% of the Internet traffic goes through mobile devices.

// Use the `responsive` mixin to apply styles
// for a particular device
.my-heading {
  font-size: 48px;
  @include responsive('mobile') {
    font-size: 24px;
  }
  // Or use the shorthand syntax:
  @include mobile {
    font-size: 24px;
  }
}
Fun fact

The composition above (with the desktop, laptop, phone & tablet graphics) is actually made using a CSS3 grid. The CSS Grid Layout Module offers a grid-based layout system, with rows and columns, making it easier to design web pages without having to use floats and positioning.
